# Gallery Image Instructions

For each image added to this folder:
1. Add the image file (e.g., Image1.png, Image2.png)
2. Create a matching text file with the same name (e.g., Image1.txt, Image2.txt)
3. The text file should contain a brief description of the image

Example:
- Image1.png
- Image1.txt (contains: "Description of what's shown in Image1")
